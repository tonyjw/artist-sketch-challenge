# Design Style Guides - Implementation Plan

## Overview
This document outlines the comprehensive plan for implementing multiple art style guides for watercolor artists. Each guide follows the structure established by the Mid-Century Modern guide.

## Guide Structure Template
Each style guide should include:
1. **Color Palettes Section** - Curated color schemes with hex codes
2. **Signature Shapes/Motifs Section** - Visual elements characteristic of the style
3. **Values & Composition Section** - Compositional principles and techniques
4. **Watercolor Techniques Section** - Medium-specific guidance
5. **Design Principles Section** - Historical context and foundational concepts

---

## 1. Japanese Sumi-e & Ink Wash Guide

### Focus
Minimalist East Asian painting traditions emphasizing simplicity, negative space, and expressive brushwork.

### Color Palettes (10-12 palettes)
- **Monochrome Variations**: 5 values of black/gray
- **Subtle Earth Tones**: Warm browns, sepia, burnt umber
- **Traditional Accents**: Vermillion red, indigo blue
- **Tea Ceremony**: Matcha green, bamboo, rice paper cream
- **Cherry Blossom**: Soft pinks with gray-greens
- **Autumn Leaves**: Rust, ochre, deep crimson
- **Mountain Mist**: Cool grays, slate blues
- **Zen Garden**: Stone grays, moss greens, sand beige
- **Ink Stone**: Charcoal to light gray gradations
- **Morning Dew**: Pale blues, misty grays

### Signature Elements
- **Four Gentlemen Strokes**: Bamboo, orchid, chrysanthemum, plum blossom
- **Brush Techniques**: Bone method, boneless method, splash ink
- **Natural Subjects**: Pine trees, cranes, koi, mountains
- **Line Quality**: Thick-to-thin, dry brush, wet pooling
- **Negative Space**: Ma (emptiness), unpainted areas

### Composition Principles
- Rule of avoiding center placement
- Asymmetrical balance
- Diagonal movement
- Heaven-earth-man triadic composition
- Breathing room and emptiness

### Watercolor Techniques
- Single-stroke leaves and petals
- Controlling water-to-pigment ratio
- Lifting and blooming for texture
- Working fast before drying
- Using absorbency of paper

### Recommended Pigments
- Neutral Tint, Payne's Gray, Lamp Black
- Burnt Umber, Raw Umber, Sepia
- Indigo, Vermillion (Pyrrol Scarlet)
- Sap Green, Hooker's Green

## 2. Botanical Illustration Guide

### Focus
Scientific accuracy meets artistic beauty - detailed plant portraits with precise color and form.

### Color Palettes (15-20 palettes)
- **Foliage Greens**: 10+ green variations from yellow-green to blue-green
- **Garden Roses**: Pinks, corals, deep reds, yellows
- **Wildflower Meadow**: Purple, yellow, white, orange mixes
- **Succulent Garden**: Blue-greens, dusty pinks, sage
- **Autumn Botanicals**: Russet, gold, deep burgundy
- **Spring Blossoms**: Soft pinks, whites, fresh greens
- **Tropical Palette**: Vibrant reds, oranges, deep greens
- **Herb Garden**: Silvery greens, purple, soft yellows
- **Vintage Herbarium**: Muted tones, aged paper backgrounds
- **Forest Floor**: Browns, mossy greens, mushroom tans

### Signature Elements
- Leaf structures: veining patterns, serrations, compound leaves
- Flower anatomy: petals, stamens, pistils
- Growth patterns and habit
- Root systems and bulbs
- Seed pods and fruit
- Detail techniques: stippling, fine lines, gradual transitions

### Composition Principles
- Single specimen layouts
- Branch arrangements
- Life cycle progression (bud to bloom to seed)
- Scientific plate format
- Grouping by family

### Watercolor Techniques
- Layering for dimensional leaves
- Wet-on-dry for precision
- Graduated washes for petal depth
- Fine detail brushwork
- Masking for highlights
- Creating realistic shadows

### Recommended Pigments
- Green mixing: Lemon Yellow + Phthalo Blue variations
- Sap Green, Hooker's Green, Permanent Green
- Quinacridone Rose, Pyrrol Scarlet
- Yellow Ochre, Raw Sienna
- Burnt Sienna for shading

## 3. Impressionist Landscape Guide

### Focus
Capturing fleeting light and atmosphere through broken color and loose brushwork.

### Color Palettes (12-15 palettes)
- **Golden Hour**: Warm oranges, yellows, long purple shadows
- **Blue Hour**: Cool blues, purples, pink highlights
- **Midday Summer**: Bright greens, clear blues, white highlights
- **Overcast Day**: Muted grays, soft greens, diffused light
- **Autumn Field**: Golds, russets, burnt oranges
- **Spring Meadow**: Fresh greens, lavender, soft yellows
- **Water Reflections**: Blues, greens with broken color
- **Snowy Landscape**: Cool whites, blue shadows, warm highlights
- **Sunset Sky**: Coral, peach, purple, gold gradations
- **Morning Mist**: Soft grays, pale yellows, cool blues

### Signature Elements
- Broken color technique
- Visible brushstrokes
- Light and shadow patterns
- Atmospheric perspective
- Reflections in water
- Movement in clouds and foliage
- En plein air subject matter

### Composition Principles
- Horizon line placement (rule of thirds)
- Foreground-middleground-background depth
- Leading lines (paths, rivers, fences)
- Atmospheric perspective (cooler/lighter in distance)
- Color temperature shifts

### Watercolor Techniques
- Wet-in-wet for skies
- Dry brush for texture
- Optical mixing on paper
- Lifting for clouds
- Quick gestural strokes
- Plein air speed painting

### Recommended Pigments
- Ultramarine Blue, Cerulean Blue
- Sap Green, Viridian
- Cadmium Yellow Light, Yellow Ochre
- Quinacridone Rose, Permanent Alizarin Crimson
- Burnt Sienna, Raw Umber

## 4. Art Nouveau / Arts & Crafts Guide

### Focus
Organic elegance from 1890s-1910s featuring flowing lines, nature motifs, and decorative beauty.

### Color Palettes (12-15 palettes)
- **Mucha Jewels**: Muted golds, dusty roses, sage greens, soft blues
- **Peacock Feather**: Teals, purples, golds, emerald greens
- **Wisteria Vine**: Lavenders, purples, soft greens, cream
- **Art Nouveau Poster**: Terracotta, olive, mustard, burgundy
- **William Morris**: Deep reds, forest greens, gold accents
- **Glasgow Rose**: Soft pinks, pale greens, silver grays
- **Tiffany Glass**: Rich blues, greens, ambers, ruby reds
- **Belle Époque**: Champagne, rose, mint, gold
- **Iris Garden**: Deep purples, yellows, blue-greens
- **Copper & Patina**: Verdigris greens, copper orange, aged browns

### Signature Elements
- Whiplash curves and flowing S-curves
- Organic motifs: poppies, irises, peacock feathers, vines
- Elongated figures and forms
- Decorative borders and frames
- Stylized natural forms
- Interlacing patterns
- Hair as decorative element
- Floral medallions

### Composition Principles
- Symmetrical vs asymmetrical layouts
- Vertical emphasis
- Integration of text and image
- Negative space as pattern
- Border treatments
- Flat pattern vs dimensional rendering

### Watercolor Techniques
- Flat washes with hard edges
- Gradual color transitions
- Metallic accents (gold, copper)
- Line and wash combination
- Decorative detail work
- Layering transparent colors

### Recommended Pigments
- Quinacridone Gold, Raw Sienna
- Quinacridone Rose, Opera Rose
- Viridian, Phthalo Green
- Ultramarine, Cobalt Blue
- Burnt Umber for outlines
- Metallic watercolors for accents

## 5. Contemporary Abstract Expressionism Guide

### Focus
Bold gestural painting emphasizing emotion, spontaneity, and the physical act of painting.

### Color Palettes (10-15 palettes)
- **Primary Power**: Bold reds, yellows, blues with black
- **Earth & Fire**: Burnt oranges, umbers, charcoal, rust
- **Ocean Depths**: Deep blues, teals, whites, grays
- **Sunset Abstract**: Oranges, pinks, purples, yellows
- **Monochrome Drama**: Black, white, grays
- **Jewel Tones**: Emerald, sapphire, ruby, amethyst
- **Pastel Softness**: Pale pinks, blues, yellows, lavenders
- **Industrial**: Grays, rust, black, muted blues
- **Nature Abstracted**: Greens, browns, sky blues
- **Neon Pop**: Bright pinks, oranges, yellows, teals

### Signature Elements
- Gestural marks and drips
- Color fields
- Layered transparent washes
- Splatter and spray techniques
- Bold contrasts
- Energetic line work
- Texture building
- Happy accidents and blooms

### Composition Principles
- All-over composition
- Balance through color weight
- Tension and release
- Movement and energy
- Edge-to-edge painting
- Asymmetrical balance
- Focal points through contrast

### Watercolor Techniques
- Wet-in-wet for blooms
- Salt textures
- Alcohol effects
- Splattering and dripping
- Lifting with spray bottles
- Masking for white spaces
- Layering transparent glazes
- Experimental techniques

### Recommended Pigments
- Phthalo Blue, Ultramarine
- Quinacridone Magenta, Pyrrol Red
- Hansa Yellow, Quinacridone Gold
- Payne's Gray, Lamp Black
- Granulating pigments for texture
- Staining pigments for intensity

## 6. Urban Sketching Guide

### Focus
On-location architectural and street scene painting with speed, spontaneity, and personal interpretation.

### Color Palettes (8-12 palettes)
- **City Morning**: Soft blues, peachy light, gray buildings
- **Afternoon Cityscape**: Warm browns, cool shadows, blue sky
- **Evening Streets**: Warm street lights, cool shadows, dusk blues
- **Rainy Day**: Grays, muted colors, wet pavement reflections
- **Brick & Stone**: Terra cotta, gray stone, weathered browns
- **Metro Interior**: Artificial lights, neutral tones
- **Market Scene**: Colorful awnings, produce colors, crowd tones
- **Industrial**: Rust, concrete gray, aged metal
- **Café Culture**: Warm wood tones, coffee browns, outdoor seating
- **Historic District**: Aged yellows, terracotta roofs, stone grays

### Signature Elements
- Simplified perspective
- Loose architectural details
- People and vehicles as indicators of scale
- Foreground, middleground, background
- Ink and wash combination
- Gestural figures
- Selective detail vs. suggestion
- Personal style and interpretation

### Composition Principles
- Eye-level perspective
- One or two-point perspective simplified
- Cropping for interest
- Focal point through detail
- Leading lines
- Framing elements
- Storytelling through scene selection

### Watercolor Techniques
- Limited palette for portability
- Quick washes
- Wet-on-dry for control
- Selective detail
- Saving whites
- Mixing on paper
- Working light to dark
- Pen and wash combination

### Recommended Pigments
**Minimal Travel Palette (6-8 colors)**:
- Ultramarine Blue or Prussian Blue
- Burnt Sienna or Burnt Umber
- Yellow Ochre or Raw Sienna
- Quinacridone Rose or Permanent Alizarin
- Sap Green or Viridian
- Payne's Gray or Neutral Tint
- Optional: Cadmium Yellow, Cerulean Blue

## 7. Minimalist Geometric Abstraction Guide

### Focus
Clean, modern restraint with hard edges, limited palettes, and mathematical harmony.

### Color Palettes (10-15 palettes)
- **Mondrian Primary**: Red, yellow, blue, black, white
- **Bauhaus**: Black, white, red, yellow, blue
- **Constructivist**: Red, black, white, gray
- **Nordic Minimal**: White, gray, black, muted blue
- **Desert Modern**: Terra cotta, sand, black, white
- **Ocean Minimal**: Navy, white, sand, pale blue
- **Forest Minimal**: Dark green, white, charcoal, tan
- **Blush & Gray**: Soft pink, grays, white
- **Mustard & Teal**: Mustard yellow, teal, cream, charcoal
- **Monochrome**: 3-5 values of single hue

### Signature Elements
- Rectangles and squares
- Circles and perfect curves
- Hard edges
- Flat color fields
- Grid structures
- Negative space as element
- Repetition with variation
- Mathematical proportions

### Composition Principles
- Golden ratio
- Rule of thirds
- Symmetry and asymmetry
- Balance through color weight
- Breathing room
- Edge relationships
- Tension between elements

### Watercolor Techniques
- Flat washes
- Masking tape for edges
- Masking fluid for whites
- Wet-on-dry layers
- Careful planning
- Clean edges
- Color blocking
- Precision work

### Recommended Pigments
- Primary colors: Hansa Yellow, Pyrrol Red, Phthalo Blue
- Neutrals: Payne's Gray, Lamp Black
- Non-granulating pigments for flat application
- Staining colors for intensity

## 8. Atmospheric Watercolor Guide

### Focus
Soft, dreamy, ethereal approaches emphasizing mood, light, and emotional resonance.

### Color Palettes (12-15 palettes)
- **Misty Morning**: Pale blues, soft grays, hints of pink
- **Fog Over Water**: Cool grays, muted blues, white
- **Dreamy Sunset**: Peach, lavender, soft pink, pale yellow
- **Forest Mist**: Muted greens, gray-blues, soft browns
- **Winter Atmosphere**: Cool whites, ice blues, lavender shadows
- **Underwater**: Turquoise, teal, pale blue, soft greens
- **Cloud Study**: Various grays, whites, hints of blue/pink
- **Moonlight**: Silver grays, cool blues, soft purples
- **Desert Haze**: Dusty pinks, pale blues, warm tans
- **Rain Clouds**: Dark to light grays, soft blues

### Signature Elements
- Soft edges
- Gradual transitions
- Layered transparent washes
- Granulation textures
- Blooms and backruns (controlled)
- Vignettes
- Diffused light
- Atmospheric depth

### Composition Principles
- Center of interest through value contrast
- Soft edges except focal area
- Depth through layering
- Minimal detail, maximum mood
- Color harmony
- Gradual value transitions

### Watercolor Techniques
- Wet-in-wet mastery
- Graded washes
- Lifting techniques
- Granulating pigments
- Multiple transparent layers
- Controlled blooms
- Soft edge work
- Negative painting for depth
- Glazing

### Recommended Pigments
**Granulating pigments for texture**:
- Cerulean Blue, Cobalt Blue, Ultramarine
- Burnt Sienna, Raw Umber
- Manganese Blue Hue
- Lunar Earth series

**Transparent pigments for layering**:
- Quinacridone Rose, Quinacridone Gold
- Phthalo Blue, Phthalo Green
- Permanent Alizarin Crimson

## Technical Implementation Notes

### File Structure
- Main file: `design-style-guides.html`
- Each style's data embedded in JavaScript
- Style switcher updates all sections dynamically
- Preserve existing MCM content as first/default style

### JavaScript Structure
```javascript
const styleGuides = {
  'mid-century-modern': { /* existing MCM data */ },
  'japanese-sumi-e': { /* new data */ },
  // etc.
}
```

### Required Data for Each Style
- `palettes`: Array of palette objects with name, description, colors
- `shapes`: Array of shape/motif objects with name, desc, SVG
- `compositions`: Array of composition examples
- `techniques`: Content for techniques section
- `principles`: Content for principles section
- `title`: Display name
- `subtitle`: Descriptive tagline

### Responsive Design
- Maintain existing responsive patterns
- Ensure dropdown works on mobile
- Keep tab navigation functional
- Preserve print-friendly aspects
